## 1. プロジェクト概要

### 目的
AI動画生成（Kling / CreaAI）×SNSマーケティングをテーマにしたZOOMセミナーを開催し、
**ISAIのAIクリエイターコースへの申込 or 個別相談への誘導**を最大化する。

### ターゲット
40〜60代のサラリーマン・主婦・体力的に限界の自営業者
→「うだつのあがらない層」が対象。経営者・管理職ではない。

### セミナー基本仕様
| 項目 | 内容 |
|------|------|
| 告知上の時間 | **90分**（長いと申込率が下がるため） |
| 実質時間 | **120分〜（オープンエンド）** |
| スライド枚数 | **約200枚**（1スライド＝1メッセージ／30秒目安） |
| 形式 | オンラインZOOM ウェビナー |
| 講師 | 未定（テンプレート状態で納品済み） |

## 5. スライドの更新方法

### スライド内容を変える場合
HTMLファイルをテキストエディタで開き、`const slides = [` から始まるJavaScript配列を編集する。

```javascript
// 例：スライドオブジェクトの構造
{
  h: 'メインテキスト（見出し）',
  s: 'サブテキスト（本文）',
  c: 'スライドタイプ',           // 下記タイプ一覧参照
  items: ['項目1', '項目2'],      // リスト・比較スライドで使用
  note: 'スピーカーノート（Nキーで表示）',
  badge: 'バッジラベル',
}
```

### スライドタイプ一覧
| タイプ (`c:`) | 用途 | 外観 |
|--------------|------|------|
| `wait` | 待機画面 | 紺地・白文字 |
| `title` | セクションタイトル | グレー地・大文字 |
| `impact` | 衝撃数字・一言 | グレー地・超大文字 |
| `impact orange` | 強調インパクト | オレンジ文字 |
| `impact red` | 恐怖・危機感 | 赤文字 |
| `content` | 見出し＋本文 | 標準レイアウト |
| `list` | 箇条書き | チェックマーク付き |
| `chat` | チャット誘導 | オレンジ左ボーダー |
| `demo` | デモ画面 | 青地・破線枠 |
| `quote` | 受講生の声 | 引用符・イタリック |
| `compare` | Before/After比較 | 2カラム（赤/緑） |
| `stack` | 特典スタッキング | 累積チェックリスト |
| `teaser` | 特典ティザー | 紺地・オレンジ文字 |
| `reveal` | 特典リビール | 白地・通常 |
| `value` | 価値・金額 | 大きな金額表示 |
| `price` | 価格プラン | 取消線つき価格 |
| `cta` | 申込CTA | オレンジ地・白文字 |
| `password` | パスワード発表 | 紺地・超大文字 |
| `qa` | Q&Aカード | 質問/回答形式 |
| `guarantee` | 返金保証 | 緑地・盾アイコン |
| `plan-table` | 松竹梅プラン表 | 3カラム比較 |
| `ending` | エンディング | 紺地・白文字 |

## 7. セミナー構成（5フェーズ）

```
Phase 0: プレ・ウェビナー（開始15分前〜）
  → 接続確認、チャットのハードルを下げる

Phase 1: 導入・権威の確立（0〜15分）
  → 衝撃デモ → Big Promise → パスワード1「3」→ ルール → 講師ストーリー
  → 「3つのこと」（逆説・謎かけ型）で最後まで残る理由を作る

Phase 2: 教育「3つの秘密」（15〜55分）
  秘密①：今動かないことが最大のリスク（40-60代が有利な理由も）
  秘密②：経験ゼロがプロより有利な理由＋Klingライブデモ（1回のみ）
  秘密③：稼げる人と稼げない人の唯一の違いは「環境」

Phase 3: トランジション（55〜60分）
  → 独学 vs 環境の2択 → 許可取り

Phase 4: オファー・スタッキング（60〜85分）
  → ISAIの紹介 → 特典を1個ずつドラマチックに発表（4特典）
  → 松竹梅プラン → ROI → 全額返金保証 → CTA

Phase 5: Q&A・クロージング（85分〜終了）
  → Q&A（35分以上使う） → 購入者読み上げ → クロージング
```

## 8. 設計の核心ポイント（DRM）

### 「すげえ」の散りばめ方
**セミナーの成約率は「うおー、すげえ！やりたい！やらなきゃ！」の総量で決まる。**
5〜7分に1回、テンションが上がるポイントを設ける（目標：18〜20ポイント）

### 先出し vs 初出し
| 分類 | 教育パート（先出し） | オファーパート（初出し） |
|------|-------------------|---------------------|
| Klingデモ | ライブで体験させる | 「さっきのツールが全部使える」で回収 |
| 受講生事例 | 成果物を見せる | 「この人たちと同じ環境に入れる」 |
| レシピ集等 | 出さない | 1個ずつドラマ付きで初出し |
| 返金保証 | 出さない | 安心のサプライズ |

### チャット設計の原則（改善済み）
- ❌ 「チャットに『○○』と打ってください」（命令形 → テクニック感が出る）
- ✅ 「正直に教えてください」「どう感じましたか？」（本音引き出し型）

### デザインレベル：Level 3（意図的に「手作り感」）
プロのデザイナーが作ったPPTは逆に売れない。
「友達が一生懸命作ったパワポ」がこのターゲット層には最適。
UGCスタイルは購買意思決定への影響がプロ制作の9.8倍（Stackla調査）
